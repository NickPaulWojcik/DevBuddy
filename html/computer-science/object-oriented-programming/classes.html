What are classes?

Classes are the blue prints of objects and are files that end in the class file type. They hold the information used to create objects when a developer instantiates one. Classes consist of four major components:

  • Import statements: Import statements import other classes into the current class for use
  • Variable declarations: Variable statements create variables for use in the current class
  • Constructor functions: Constructor functions are the functions developers call to create new objects
  • Functions: Functions are the actions of the class, they can use the classes variables and other functions

How are classes used?

Classes are created when developers need a new object to use. They will create a class with all the major components needed and then create objects from it.
